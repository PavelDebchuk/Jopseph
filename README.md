## Getting Started

**Install dependencies**

```bash
yarn run start
```

**Start backend only use:**

```bash
yarn run server
```

**Start frontend only use:**

```bash
yarn run app
```

**Start dev environment use:**

```bash
npx nodemon
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
