
## Getting Started

**Install dependencies**

```bash
yarn run start
```

**Start backend only use:**

```bash
yarn run server
```

**Start frontend only use:**

```bash
yarn run app
```

**Start dev environment use:**

```bash
yarn run start:dev
```

Open [http://localhost:3005](http://localhost:3005) with your browser to see the result.
